# Nier Automata Style AI Chat Application

This is an AI-powered chat application that allows users to interact with locally downloaded AI models. The application features a Nier Automata inspired design and allows users to choose their AI model at startup.

## Features

- Choose from multiple locally downloaded AI models at startup
- Nier Automata themed interface
- Local AI model integration
- Conversational AI experience

## Requirements

- Python 3.8+
- Required Python packages (see requirements.txt)
- Downloaded AI models

## Installation

1. Clone this repository
2. Install required packages: `pip install -r requirements.txt`
3. Download your preferred AI models
4. Configure model paths in the settings

## Usage

Run the application:
```
python main.py
```

Follow the on-screen instructions to select your AI model and start chatting.

## Project Structure

- `main.py` - Main application entry point
- `models/` - Directory for AI model configurations
- `ui/` - User interface components
- `assets/` - Game assets and styling
- `config/` - Configuration files
